# What I’m building

I’m building an interactive geography game where users have 15 minutes to name all 197 countries in the world, entering answers in any order while filling in a clean and structured interface.


# Who it’s for / Why:

This game is for geography enthusiasts, students, and anyone who enjoys quiz-based learning. I recently started playing geography quizzes and found that I really enjoy them. I can now name all 197 countries. However, I noticed that existing platforms can feel visually cluttered or not very intuitive to use. This tool aims to create a cleaner, more engaging experience where users can focus on recalling countries without being constrained by layout or order.


# Core Features:

1. 15-minute countdown timer that updates in real time
2. Free text input system where users can guess countries in any order
3. Dynamic updates that replace placeholders with correct country names
4. Progress tracker showing how many countries have been guessed out of 197
5. Reset / new game button to restart without refreshing


# What I don’t know yet:

How to store and manage all 197 country names efficiently in JavaScript
How to handle user input matching (case sensitivity, duplicates, slight variations)
How to implement a real-time countdown timer that updates the page
How to dynamically update a large number of elements (197 placeholders) smoothly
